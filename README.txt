Follow these steps to decrypt cyber1.txt and cyber2.txt.

1. Install the cryptography python package. In the command prompt, type ['py -m pip install cryptography']. If PIP is not installed, precede this command by typing ['py get-pip.py']

2. Download these two python files.

3. Put the two python files, and the two encrypted cyber1.txt and cyber2.txt files into the same folder. Then run keyproduction.py , entering my UCAS personal ID when prompted.

4. Next, run decrypter.py .

5. Read the new decrypted files, cyber1DECRYPTED.txt and cyber2DECRYPTED.txt .
